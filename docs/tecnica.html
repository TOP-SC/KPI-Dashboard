El dashboard principal muestra tres bloques alineados al pizarrón inicial. Los totales vienen en resumen.pizarra del JSON de la API.

Bloque UICampo APISignificado

Valor conocimiento interno

KPI de Comparativos: Discovery TOP acumulado. Monetiza el conocimiento lógico del negocio que TOP incorpora y un externo no trae incluido. 

Por caso:   valor_conocimiento = discovery_horas × TARIFA_INVERSION (50 USD/h)
Portfolio:  valor_conocimiento_interno_usd = Σ discovery_horas_top × 50

Horas discovery:
  1. Columna sheet "Discovery (hs)" si existe
  2. Si no → estimación por complejidad (score del contexto del caso)

Comparativa contextual

Motor en backend/services/comparativa_contexto.py. Clasifica cada caso y activa solo los componentes de costo que aplican a la alternativa externa.

ModalidadCuándoComponentes típicos empresa/app
script_ligeroAlertas, VBA, Apps Script <24hDev + discovery (sin fee/impl)
desarrollo_medidaDefault customDev + discovery + impl + fee + nube
integracion_enterpriseSAP, Comex, ERP, apps enterprise+ mantenimiento; app + impl + mant
saas_encajeApp SaaS encaja, baja complejidadLicencia + brecha reducida

API por caso: comparativo.contextual.externos (desglose) · comparativo.modo_label · totales en freelancer_total_usd, etc.

Catálogo v2: modelos_default.saas vs modelos_default.enterprise en data/catalogo_mercado.json (implementacion_pct, mantenimiento_pct_anual, brecha_factor).

Fórmulas KPI (por caso)

Inversión TOP

inversion_usd = horas_invertidas × TARIFA_INVERSION_USD_H
(default: USD 50/h)

Ahorro anual

horas_ahorradas_anual = reduccion_hs_mes × 12
ahorro_anual_ars = horas_ahorradas_anual × TARIFA_AHORRO_ARS_H
ahorro_anual_usd = ahorro_anual_ars / cotizacion_dolar
(cotización: dolarapi blue si COTIZACION_AUTO=true)

ROI

roi_usd = ahorro_anual_usd − inversion_usd

Comparativo Freelancer

discovery_externo = discovery_top × 1.5
freelancer_total = (hs_dev × USD 75/h) + discovery_externo

Comparativo Empresa

empresa_total = (hs_dev × USD 100/h) + discovery_externo

App mercado real

licencia_horizonte = usd_anual_catálogo × 3 años
brecha = discovery_freelancer × (1 + 35%) × factor_conocimiento
app_costo_real = licencia_horizonte + brecha

App y precio desde data/catalogo_mercado.json, no columnas Drive.

Instalación y arranque

Requisitos

  • ✓Python 3.11+ · Node.js 18+
  • ✓Windows (iniciar.bat) — adaptable a Linux/Mac

Arranque rápido

1. Copiar .env.example → .env y ajustar parámetros
2. Doble click en iniciar.bat
   → crea venv, instala deps, compila front, abre browser
3. URL típica: http://127.0.0.1:8765

Nota: el pedido se cumple con uso local (iniciar.bat). Servidor interno es opcional.
